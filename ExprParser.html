ExprParser

Description

This module uses the Expr datatype. The parsers in this module take a string of format and parse it into an expression of corresponding type (i.e Int, Float, Double...)

Synopsis

  • parserExprD :: String -> Expr Double
  • parserExprF :: String -> Expr Float
  • parserExprI :: String -> Expr Int
  • parserExprIg :: String -> Expr Integer
  • parserVectorExprD :: String -> Expr [Double]
  • parserVectorExprF :: String -> Expr [Float]
  • parserVectorExprI :: String -> Expr [Int]
  • parserVectorExprIg :: String -> Expr [Integer]
